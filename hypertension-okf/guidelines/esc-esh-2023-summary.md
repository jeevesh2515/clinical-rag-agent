# ESC/ESH 2023 Summary

## BP Categories

| Category | Systolic | Diastolic |
|----------|----------|-----------|
| Optimal | <120 | <80 |
| Normal | 120–129 | 80–84 |
| High Normal | 130–139 | 85–89 |
| Grade 1 HTN | 140–159 | 90–99 |
| Grade 2 HTN | 160–179 | 100–109 |
| Grade 3 HTN | ≥180 | ≥110 |
| Isolated Systolic HTN | ≥140 | <90 |

## Treatment Thresholds

| Setting | Threshold |
|---------|-----------|
| Office BP | ≥140/90 |
| Home BP | ≥135/85 |
| Ambulatory (daytime) | ≥135/85 |

## BP Goals

| Population | Goal |
|------------|------|
| All treated patients | <140/90, aim for <130/80 if tolerated |
| Elderly (≥80) | <150/90, aim for <140/80 if tolerated |
| CKD, diabetes, CAD | <130/80 |

## First-Line Strategy

- Start with **single-pill combination** (ACEi/ARB + CCB or thiazide-like) in most patients
- Two-drug combination preferred over monotherapy for most patients
- Three-drug combination (ACEi/ARB + CCB + thiazide) if BP not controlled
