# JNC8 Summary (2014)

## BP Thresholds for Treatment

| Population | Threshold (systolic) | Threshold (diastolic) |
|------------|---------------------|----------------------|
| ≥60 years | ≥150 mmHg | ≥90 mmHg |
| <60 years | ≥140 mmHg | ≥90 mmHg |
| CKD or diabetes (all ages) | ≥140 mmHg | ≥90 mmHg |

## BP Goals

| Population | Goal |
|------------|------|
| ≥60 years | <150/90 |
| <60 years | <140/90 |
| CKD or diabetes (all ages) | <140/90 |

## First-Line Agents by Population

| Population | First-Line | Alternative |
|------------|------------|-------------|
| Non-black | Thiazide, CCB, ACEi, ARB | — |
| Black | Thiazide or CCB | — |
| CKD (any race) | ACEi or ARB | — |

## Algorithm

1. Start one first-line agent, titrate to max dose
2. If goal not reached, add a second agent from a different class
3. If goal still not reached, add a third agent (consider thiazide if not already used)
4. If still uncontrolled, consider referral or add spironolactone or other agent
