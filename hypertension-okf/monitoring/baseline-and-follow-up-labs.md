# Baseline and Follow-Up Labs

## Baseline Laboratory Evaluation

| Test | Rationale |
|------|-----------|
| **Serum creatinine + eGFR** | Baseline renal function; guide drug selection (ACEi/ARB/diuretic); detect CKD |
| **Serum potassium** | Baseline for ACEi/ARB, diuretic monitoring; detect primary aldosteronism (if low) |
| **Serum sodium** | Baseline for diuretic monitoring |
| **Fasting glucose or HbA1c** | Screen for diabetes (common comorbidity) |
| **Lipid panel** | ASCVD risk calculation |
| **Urinalysis** | Detect proteinuria, hematuria (CKD, glomerular disease) |
| **ECG** | Detect LVH, prior MI, arrhythmia |
| **Uric acid** | If gout suspected; baseline for diuretic therapy |
| **Calcium** | Baseline; rule out hyperparathyroidism |
| **TSH** | If symptoms or atrial fibrillation (hyperthyroidism → HTN) |

## Follow-Up Lab Schedule

| Test | Frequency | Trigger for Earlier Check |
|------|-----------|--------------------------|
| Creatinine / eGFR | 1–2 weeks after starting or titrating ACEi/ARB/diuretic; then q6–12 months | >30% increase from baseline |
| Potassium | 1–2 weeks after starting or titrating ACEi/ARB/diuretic; then q6–12 months | ACEi/ARB + CKD + K-sparing diuretic |
| Sodium | 1–2 weeks after starting diuretic; then annually | Elderly on thiazide (hyponatremia risk) |
| HbA1c | Annually | If prediabetes or diabetes |
| Uric acid | If new-onset gout | After starting diuretic |
| Albumin:creatinine ratio (ACR) | Annually (if CKD or diabetes) | BP not at target |
| Lipid panel | Annually | To recalculate ASCVD risk |

## When to Add Additional Testing

| Clinical Scenario | Additional Test |
|------------------|----------------|
| Hypokalemia (K+ <3.5) not on diuretic | Aldosterone:renin ratio (screen for primary aldosteronism) |
| Resistant HTN | Aldosterone:renin ratio, sleep study, renal imaging |
| Paroxysmal HTN + headache/palpitations | Plasma metanephrines (pheochromocytoma) |
| Abdominal bruit or young-onset HTN | Renal artery duplex/CTA (fibromuscular dysplasia in young women) |
| Creatinine rise >30% after ACEi/ARB | Renal duplex (consider bilateral RAS) |
