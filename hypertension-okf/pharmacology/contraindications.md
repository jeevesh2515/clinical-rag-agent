# Contraindications Per Drug Class

## ACE Inhibitors

| Contraindication | Severity | Alternative |
|-----------------|----------|-------------|
| **Pregnancy** | Absolute | Labetalol, Nifedipine, Methyldopa |
| **History of angioedema** | Absolute | ARB (low risk but not zero) |
| **Bilateral renal artery stenosis** | Absolute | CCB or beta-blocker |
| **Hyperkalemia (K+ >5.5)** | Relative, withhold | Switch to alternative |
| **Severe aortic stenosis** | Relative caution | Monitor |
| **Cough (intolerable)** | Relative (switch class) | ARB (no cough) |

## ARBs

| Contraindication | Severity | Alternative |
|-----------------|----------|-------------|
| **Pregnancy** | Absolute | Labetalol, Nifedipine, Methyldopa |
| **Bilateral renal artery stenosis** | Absolute | CCB or beta-blocker |
| **Hyperkalemia (K+ >5.5)** | Relative, withhold | Switch to alternative |
| **History of ACEi angioedema** | Relative (very low risk) | Consider, but counsel patient |

## Calcium Channel Blockers

| Contraindication | Severity | Notes |
|-----------------|----------|-------|
| **Heart block (2nd/3rd degree)** | Absolute (non-DHP) | Use DHP CCB instead |
| **HFrEF (EF <40%)** | Relative (non-DHP) | Diltiazem/verapamil are negative inotropes; amlodipine is safe |
| **Severe aortic stenosis** | Relative (all CCBs) | Use with caution |
| **Liver cirrhosis** | Relative (all CCBs) | Reduce dose |
| **Concomitant beta-blocker** | Relative (non-DHP) | Additive bradycardia |

## Thiazide Diuretics

| Contraindication | Severity | Alternative |
|-----------------|----------|-------------|
| **Gout (active)** | Absolute | ACEi, ARB, or CCB |
| **Anuria** | Absolute | — |
| **Severe hyponatremia (<125)** | Relative, avoid | Use alternative class |
| **eGFR <30** | Relative (ineffective) | Loop diuretic |
| **Symptomatic hypercalcemia** | Relative, avoid | Use alternative class |

## Beta-Blockers

| Contraindication | Severity | Alternative |
|-----------------|----------|-------------|
| **Asthma (active)** | Absolute | CCB, ACEi |
| **Heart block (2nd/3rd degree)** | Absolute | CCB (DHP) |
| **Sick sinus syndrome (no pacemaker)** | Absolute | CCB (DHP) |
| **Severe bradycardia (<50)** | Relative, use with caution | Alternative class |
| **COPD (severe)** | Relative | Avoid unless cardioselective |
| **Psoriasis** | Relative | Avoid |
