# Resistant Hypertension Workup

## Definition

Resistant hypertension is defined as BP **above goal** despite concurrent use of **three antihypertensive agents** from **different classes**, ideally including:

- A **thiazide or thiazide-like diuretic** (e.g., chlorthalidone, indapamide)
- A **CCB** (e.g., amlodipine)
- An **ACEi or ARB**

All agents should be at **maximum tolerated doses**.

**Pseudoresistance** must be excluded before labeling as resistant.

## Step 1: Exclude Pseudoresistance

| Cause | How to Assess |
|-------|---------------|
| Poor medication adherence | Pill counts, pharmacy refill data, motivational interviewing |
| White-coat effect | Home BP monitoring or ambulatory BP monitoring (ABPM) |
| Improper BP technique | Cuff size, rest period, seated vs standing |
| Suboptimal regimen | Not using max doses, missing drug classes |
| Lifestyle factors | High sodium intake, obesity, alcohol excess |

## Step 2: Evaluate for Secondary Causes

| Cause | Screening Test |
|-------|---------------|
| Primary aldosteronism | Aldosterone:renin ratio |
| Chronic kidney disease | eGFR, ACR, renal ultrasound |
| Renal artery stenosis | Renal duplex ± CTA/MRA (young-onset, worsening, flash pulmonary edema) |
| Obstructive sleep apnea | STOP-BANG, nocturnal oximetry, polysomnography |
| Pheochromocytoma | Plasma metanephrines |
| Cushing syndrome | Overnight dexamethasone suppression, 24h urinary cortisol |
| Coarctation of aorta | Brachial-femoral delay, imaging |

## Step 3: Pharmacologic Management

### Preferred 4th-Line Agent

| Agent Class | Examples | Notes |
|------------|----------|-------|
| **Mineralocorticoid receptor antagonist** | **Spironolactone** 25–50 mg | Most evidence; gold standard 4th agent |
| **MRA alternative** | **Eplerenone** 50–100 mg | If spironolactone side effects |
| **Alpha-blocker** | **Doxazosin** 4–8 mg | If MRA not tolerated or GFR <30 |
| **Beta-blocker** | **Bisoprolol** 5–10 mg | If heart rate >80 or CAD present |
| **Centrally acting agent** | **Clonidine** | Reserved for advanced cases |

### Step 5 and Beyond

- Add agents sequentially from different classes
- Consider **loop diuretic** (e.g., torsemide, furosemide) if eGFR <30
- Avoid combining ACEi + ARB (no added benefit, more hyperkalemia/hypotension)
- Refer to [[comorbidities/htn-and-ckd]] for CKD-specific guidance

## Step 4: Referral

Refer to hypertension specialist if:

- BP remains above goal on **≥4 agents** after 6 months
- Suspected secondary cause requires specialized testing
- Intolerance to multiple drug classes
- Hypertensive emergency with target organ damage
