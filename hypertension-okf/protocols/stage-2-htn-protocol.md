# Stage 2 HTN Step-Up Treatment

## Definition

- **ACC/AHA**: BP ≥140/90 mmHg
- **NICE**: Clinic BP ≥160/100 (or ABPM/HBPM ≥150/95)
- **ESC/ESH**: Office BP 160–179 / 100–109 mmHg

## When to Escalate

- BP remains ≥140/90 (or ≥130/80 for high-risk) after max-tolerated dual therapy
- Confirm with 2-week home BP log before escalating
- Exclude non-adherence, white coat effect, and suboptimal regimen first

## Escalation Steps

| Step | Add | Example | Notes |
|------|-----|---------|-------|
| **1st line** | Dual therapy | ACEi/ARB + CCB | Start as single-pill combination if possible |
| **2nd line** | Triple therapy | ACEi/ARB + CCB + **thiazide-like diuretic** | Chlorthalidone 12.5–25 mg or Indapamide 1.25–2.5 mg |
| **3rd line** | Fourth agent | Add **Spironolactone** 25–50 mg | Check K+ ≤4.5 before starting; monitor K+, consider gynecomastia |
| **Alternative to spironolactone** | Alpha-blocker | Doxazosin 4–8 mg | If K+ >4.5 or spironolactone not tolerated |
| **Alternative** | Beta-blocker | Bisoprolol 5–10 mg or Atenolol 25–50 mg | Not first-line unless CAD, HF, or specific indication |

## When to Refer

| Criteria | Referral |
|----------|----------|
| BP ≥180/110 despite triple therapy | Urgent specialist referral |
| Suspected secondary HTN | Routine specialist referral |
| Spironolactone contraindicated/not tolerated | Discuss with specialist |
| Stage 2 + CKD stage 4+ | Nephrology referral |
| Stage 2 + hypokalemia | Check aldosterone:renin ratio |
